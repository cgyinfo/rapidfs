# CGYINFO: RapidFS

## Statement

rapidfs is a http file upload service.

## Usage

rapidfs is a nodejs application service for http file uploading.
System running environment requirements:

-   nodejs >= 12.14
-   npm >= 6.13

```
node index.js
```
